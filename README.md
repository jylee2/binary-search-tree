# Binary Search Tree

Create a binary search tree with the use of a primary key.

```
npm install git://github.com/jylee2/binary-search-tree.git

/*
@params
- Example dataArray: [
    { name: "batman", powers: ["rich"] },
    { name: "aquaman", powers: ["swimming"] },
  ]
- Example primaryKey: "name"
*/
```

# Run

`npm i`
`node index.js`

# Test

`npm run test`
